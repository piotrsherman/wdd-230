WebWeather Site Plan

Peter Herman

Site Name

WebWeather (webweather.org)

Site Purpose

The purpose of this site is to inform users about the weather, as well as so that the user can easily get the information he needs on the site.

Audience

The main audience of the site are all those people who visit the site using a computer or smartphone in order to find out the weather either for the next few days or for the current week.
