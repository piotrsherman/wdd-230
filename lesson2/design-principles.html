microsoft.com

I liked this site as an example of a visual hierarchy because we can see on it that some elements are highlighted brightly and others are secondary, and so on. Depending on the importance of the information for the user according to the author's provision.

Repetition

Google Fonts

fonts.google.com/about

I chose this site as a good example of repetition. We can notice that styles are usually repeated on it, since the goal is to allow the user to concentrate on the text and so that the contrast of different elements does not unfocus the user's attention.
